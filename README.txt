Simple Create, Read, Update, Delete Java application using Maven, Spring MVC, MySQL, Tomcat, Hibernate frameworks.

Install order:
1)Open project with your IDE.
2)Import database to MySQL using TestBase.sql file in root directory. Database with name "test" will be created from scratch or rewrite existing database with a same name.
3)Connect created database to project. Database login and password are the same - "root".
4)In folder "src\main\resources" open the file "application.properties". In this file, change values of keys "database.username" and "database.password" according to your connection data in MySQL.
5)Connect Tomcat to project and run it.
6)Enjoy!
